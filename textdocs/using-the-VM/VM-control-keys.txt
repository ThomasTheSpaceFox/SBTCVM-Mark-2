VM control keys:

[ESCAPE]:
issues a User stop.
The VM will wait for you to press enter to exit

[F2]:
Toggles STEP-BY-STEP mode.
also can be used as a "pause" feature

[F7]:
dump raster fields to CAP directory. 
Same filenames will be used each time,
so be sure to rename/move images you wish to keep.

[F8]:
Take a screen shot and save it into CAP directory.


[F10]:
DUMP current state of IObus and Executable bus MEMORY
(the memoryspace troms are loaded into) into *.dmp text
files in the CAP directory. 
Same filenames will be used each time,
so be sure to rename/move dumps you wish to keep.
